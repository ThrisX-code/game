# Pong Game

A classic Pong game built with HTML, CSS, and JavaScript!

## Features

✨ **Player Controls:**
- Move left paddle with **Mouse Y position**
- Move left paddle with **Arrow Keys (↑/↓)**
- Press **SPACE** to start/pause the game
- Press **R** to reset the game

🤖 **Computer AI:**
- Right paddle controlled by intelligent AI
- Follows the ball with adjustable difficulty

⚽ **Game Mechanics:**
- Bouncing ball physics
- Paddle collision detection with ball spin
- Wall collision detection
- Score tracking system
- Ball resets after each point

🎨 **Visual Design:**
- Retro neon green aesthetic
- Live scoreboard display
- Center dividing line
- Clear instructions panel
- Smooth animations

## How to Play

1. Open `index.html` in your web browser
2. Press **SPACE** to start the game
3. Use your mouse Y position or arrow keys to move the left paddle
4. The right paddle is controlled by the computer AI
5. Hit the ball to score points when the opponent misses
6. Press **R** to reset the game and scores

## Files

- `index.html` - Game HTML structure
- `style.css` - Game styling and layout
- `script.js` - Game logic and physics

Enjoy the game! 🎮
